## Como instalar e usar ?
<b>Clone o projeto</b></br>
`` $ git clone https://github.com/martins20/Projeto-Agenda.git ``
</br>
<b>Entre na Pasta e instale as dependencias</b></br>
`` $ cd Projeto-Agenda && yarn install ``
</br>
<b>Inicie o projeto</b></br>
`` $ yarn start `` 
</br>
